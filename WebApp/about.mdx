# Welcome!

This is a documentation guide written for Pi_oT. <br/>

>The main objective of this project is to integrate a Rasberry Pi to send data to Amazon Web Services,
>where information would be sent to a user-friendly web application written in React. The web application
works as two separate components: MQTTCon & react-app.<br/>

Please browse the menus for information!
