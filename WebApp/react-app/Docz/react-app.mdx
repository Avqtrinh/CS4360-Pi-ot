# react-app

>The other component working in junction with MQTTCon is our react-app folder, which is
responsible for the front end of the application. Implementation for the user interface, login/signup logic,
and MQTT object retrieval can be found here. <br/><br/>

>As with all react.js applications, the key entry points begin at App.js and Index.js.
We highly suggest looking at those first!
